## Overview

This project analyzes household electricity consumption data and applies machine learning techniques to understand usage patterns, forecast future electricity demand, detect anomalies, and categorize energy consumption levels.

The dataset used in this project comes from the UCI Machine Learning Repository and contains measurements of electric power consumption in one household over several years.

The project implements a complete machine learning workflow including exploratory data analysis, supervised learning for forecasting, unsupervised learning for anomaly detection and clustering, and a simple rule-based AI system for usage categorization.

---

## Dataset

Dataset: Individual Household Electric Power Consumption
Source: UCI Machine Learning Repository

```json
https://archive.ics.uci.edu/dataset/235/individual+household+electric+power+consumption
```

### Dataset Columns

* Date
* Time
* Global_active_power
* Global_reactive_power
* Voltage
* Global_intensity
* Sub_metering_1
* Sub_metering_2
* Sub_metering_3

The dataset contains more than 2 million records collected between 2006 and 2010.

## Task 1 — Exploratory Data Analysis (EDA)

Exploratory Data Analysis was performed to understand electricity consumption patterns over time.

Steps performed:

* Loaded and cleaned the dataset
* Combined Date and Time columns into a single Datetime field
* Handled missing values
* Converted power readings to numeric format

Visualizations created:

* Time series plot of Global_active_power
* Hourly electricity consumption pattern
* Daily average electricity usage

Key observations:

* Electricity usage shows fluctuations across days and seasons
* Usage is lowest during early morning hours
* Highest consumption occurs during evening hours when household activity increases

---

## Task 2 — Time Series Forecasting

A supervised machine learning model was developed to forecast the next-hour electricity consumption.

### Feature Engineering

Lag features were created using previous electricity readings:

* lag1 — previous time step
* lag2 — two time steps before
* lag3 — three time steps before

### Model

Random Forest Regressor was used to predict future power consumption.

### Evaluation Metrics

Model performance was evaluated using:

* Mean Absolute Error (MAE)
* Root Mean Squared Error (RMSE)

The predicted electricity consumption closely follows the trend of actual values, indicating that the model successfully captures temporal patterns in the data.

---

## Task 3 — Unsupervised Learning

Two unsupervised learning techniques were applied.

### Anomaly Detection

Isolation Forest was used to detect unusual electricity consumption patterns.

Detected anomalies correspond to unusually high electricity usage events that may indicate heavy appliance usage or irregular consumption behavior.

### Clustering

K-Means clustering was applied to daily average electricity usage.

The algorithm grouped days into three clusters representing different consumption levels:

* Low usage days
* Moderate usage days
* High usage days

This helps identify patterns in household electricity demand over time.

---

## Task 4 — Rule-Based AI System

A simple rule-based system was implemented to categorize electricity usage based on predicted Global_active_power values generated by the forecasting model.

### Categories

Low Usage
Medium Usage
High Usage

### Example Rules

```
If power < 1.0 → Low Usage  
If power < 2.5 → Medium Usage  
Else → High Usage
```

Each category also generates a simple recommendation to help interpret electricity consumption levels.

This demonstrates how machine learning predictions can be translated into actionable insights.
